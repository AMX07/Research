# Research

A showcase of my deep learning projects, from-scratch reimplementations, LLM experiments, and applied ML systems.


## Index of Deep Learning Projects

| Project | Description |
|---------|-------------|
| [micrograd](https://github.com/AMX07/micrograd) | Re-implementation of Karpathy's micrograd — a minimal scalar-valued autograd engine with a tiny neural net library on top. |
| [makemore](https://github.com/AMX07/makemore) | Character-level language models (bigram and MLP) trained to generate name-like text, following Karpathy's makemore series. |
| [minimalGPT](https://github.com/AMX07/AWS_BDA) | A from-scratch minimal GPT implemented in a single Jupyter notebook. |
| [digit_recognition](https://github.com/AMX07/digit_recognition) | PyTorch MNIST classifier served through a Streamlit web app deployed on AWS. |
| [notetaker](https://github.com/AMX07/notetaker) | Converts MP4 video lectures into structured markdown notes using Whisper for transcription and Claude for assembly. |
| [gpt-time-blindness](https://github.com/AMX07/gpt-time-blindness) | Proof-of-concept giving LLM chatbots temporal awareness via timestamp injection and system-prompt engineering. |
